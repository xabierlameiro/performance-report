# Contributing

This repository contains auto-generated Lighthouse performance reports. It is updated automatically by CI workflows from the source repository.

## How this works

1. CI workflow in [the-last-dance](https://github.com/xabierlameiro/the-last-dance) runs Lighthouse CI
2. HTML reports are committed to `lighthouse/` in this repository
3. GitHub Pages serves the latest report at the published URL

If you find a performance issue, please open an issue in the [source repository](https://github.com/xabierlameiro/the-last-dance).
