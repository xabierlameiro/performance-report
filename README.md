# Performance Report Archive

Auto-generated Lighthouse performance report archive for [xabierlameiro/the-last-dance](https://github.com/xabierlameiro/the-last-dance), deployed to GitHub Pages.

## What’s in this repo

The `lighthouse/` folder contains the latest Lighthouse HTML reports. It is updated automatically on every CI run.

## Viewing the reports

Open the [GitHub Pages site](https://xabierlameiro.github.io/performance-report/) to browse the performance audit results.

## Metrics tracked

- ⚡ Performance score
- ♿ Accessibility score
- 👍 Best Practices
- 🔍 SEO score
- 📶 Progressive Web App compliance

## Source

Performance tests run in [xabierlameiro/the-last-dance](https://github.com/xabierlameiro/the-last-dance) CI pipeline.
